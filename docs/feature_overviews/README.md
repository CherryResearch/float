These files are the plain-language companion set for float's technical feature specifications. They are meant to answer the simple product question of what a feature does, why it exists, and what a person should expect from it without making the reader parse backend routes or implementation notes.

The current overviews are [chat-and-input.md](chat-and-input.md), [tools-and-actions.md](tools-and-actions.md), [memory-knowledge-and-search.md](memory-knowledge-and-search.md), [threads.md](threads.md), [calendar-tasks-and-followups.md](calendar-tasks-and-followups.md), [agent-console.md](agent-console.md), [voice-live-and-passthrough.md](voice-live-and-passthrough.md), [models-and-runtime-modes.md](models-and-runtime-modes.md), [conversations-history-and-storage.md](conversations-history-and-storage.md), and [device-sync-and-streaming.md](device-sync-and-streaming.md). Each one stays short on purpose and complements the public setup, API, and storage docs in this repository.
